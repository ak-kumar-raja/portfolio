Integration Hub — Software Compliance Platform

Amazon employees use over 20,000 productivity applications, but only 529 were listed in IT Shop — of which 417 lacked security reviews. Teams had to check multiple fragmented systems (TPS, Talos, ASCAD for security; Contract Central for legal; UBX for HR Privacy; SIM for accessibility), causing 95% of employees to abandon software requests.

Product Vision: Developed comprehensive requirements through stakeholder interviews across Legal, Security, HR Privacy, and Accessibility teams. Created detailed PRDs, user stories, and acceptance criteria for a unified compliance platform.

Delivery: Built a system connecting 500+ IT Shop applications to their compliance records across four review types. The compliance database updates every 6 hours. API infrastructure handles 1,000 concurrent requests with 99.9% uptime. Implemented phased rollout (Q2 2025 – Q1 2026) with 2-week sprint cycles.

Results: Reduced deployment time for new service teams from 3 months to 2 weeks using predefined templates. Security reviews compressed from 3 months to 1.5 months. Teams can now reuse existing compliance reviews, eliminating duplicate assessments.

Impact: Software request abandonment targeted from 95% → under 20% • Deployment time 3 months → 2 weeks • Security reviews 3 months → 1.5 months
